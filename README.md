## Organizer for drill and hex bit

# Описание проекта

Данный проект представляет собой модульный органайзер для сверл и шестигранников, собранный из комбинации готовых конструкционных деталей и деталей, напечатанных на FDM 3D-принтере. Органайзер позволяет удобно хранить сверла разных диаметров, а также легко изменять конфигурацию по мере необходимости.

Органайзер состоит из вертикальных и горизонтальных профилей, соединенных между собой напечатанными элементами, и держателей (фишек) для каждого размера сверл. Все элементы можно собирать, разбирать и переставлять, что делает органайзер гибким и удобным для пользователя.

# Материалы и комплектующие

Для сборки органайзера использовались следующие элементы:

- Алюминиевый конструкционный профиль: V20×20L
- Сухарь пазовый: М4, паз 6
- Винты: М4
- Детали, напечатанные на FDM 3D-принтере (соединительные элементы и держатели для сверл)
- Профили нарезаны на нужную длину для сборки органайзера

# Конструкция и сборка

Органайзер состоит из нескольких ключевых частей:

1) Вертикальные профили:
Два основных профиля, стоящие вертикально, служат каркасом конструкции.

2) Горизонтальные профили:
Расположены в три уровня и соединены с вертикальными профилями с помощью напечатанных деталей и сухарей. Горизонтальные профили удерживают фишки с отверстиями под сверла.

3) Фишки для сверл:

- Фишки вставляются в пазы горизонтальных профилей.
- Каждая фишка рассчитана под конкретный диаметр сверла или шестигранника.
- При необходимости их можно добавлять, удалять или менять местами.

Таким образом, органайзер модульный и адаптивный, что позволяет подстраивать его под любые наборы инструментов.
